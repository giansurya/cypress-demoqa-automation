This repository is a QA automation portfolio project using Cypress to test the DemoQA website.
It covers UI automation for Login and Register features, including positive and negative scenarios.

##  Project Overview
The purpose of this project is to demonstrate how Cypress is used for end-to-end UI testing.
The automation scripts are created based on documented manual test cases and structured using separated locators and reusable commands.

## Test Case Documentation
Manual test cases for this project are documented in Google Sheets:
